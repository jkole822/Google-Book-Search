# Google Books Search

![license](https://img.shields.io/static/v1?label=license&message=MIT&color=green&style=for-the-badge)

## Description

Search for books using the Google Books API. Users can search for books on the 'Search' page, and for each search result, information on the corresponding book is displayed along with the option to view the book on the Google Books website or `save` the book to view later on the 'Saved' page. On the 'Saved' page, the same information and option to view the corresponding Google Books site is displayed for each saved book, but the `save` option is replaced with a `delete` option to delete the book from the 'Saved' page.
